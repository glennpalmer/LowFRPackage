# LowFR

This package implements the Low-rank longitudinal factor regression (LowFR) model proposed by: G. Palmer, A.H. Herring, D.B. Dunson (2025) 'Low-rank longitudinal factor regression with application to chemical mixtures' (https://pmc.ncbi.nlm.nih.gov/articles/PMC12013532/).

If you have any questions, find bugs, etc. please reach out to glenn.palmer@duke.edu.

# Downloading and installing

To download and install the package, run the following code.

```r
install.packages("devtools")
devtools::install_github("glennpalmer/LowFRPackage")
```
Then, for a tutorial of how to fit and summarize a LowFR model, see `demo_ELEMENT.html` (download it and open in your browser) and/or the associated R markdown notebook.
